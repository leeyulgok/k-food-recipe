# K-Food 레시피 웹 사이트
한국에 대한 관심이 증가하고 있는 시대.  
외국에서도 쉽게 한국요리를 접하고 만들 수 있는 웹 사이트 개발합니다.  

## 기술스택
1. React
2. Next js 13
3. TailwindCSS
4. Node js
5. MySQL

## 접근방식
접근 방식은 Bottom-Up 방식으로 개발을 진행했습니다.
